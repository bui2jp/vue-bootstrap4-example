# vue-bootstrap4

Vue, Bootstrap を利用したログイン画面とダッシュボード画面の簡単なサンプル

Bootrapの画面は
https://getbootstrap.com/docs/4.5/examples/dashboard/
を参考に作成

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files

```
npm run lint
```
